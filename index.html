Software libre en la educación

Por Carlos Ochoa

¿Qué es el software libre?

Software que una vez obtenido, puede ser usado, estudiado, modificado y distribuido libremente.

Libre != Gratis

Libertades del software libre

  1. La libertad de usar el programa con cualquier propósito.
  2. La libertad de estudiar como funciona el programa y adaptarlos a sus necesidades.
  3. La libertad de distribuir copias
  4. La libertad de mejorar el programa y hacer públicas las mejoras a los demas.

Software libre y propietario

Relación Software Libre y Educación

¿Por qué usar software libre en la Educación?

  • Crear profesionales independientes de un determinado entorno de software.
  • Reduce costos.
  • Permite que los alumnos puedan usar el mismo software con el que se enseña.
  • Ofrece control sobre el software.

Tipos de software

  • Software de sistemas (Sistemas Operativos).
  • Software de aplicación.
  • Software de programación.

Sistemas Operativos

El Sistema Operativo es aquel que proporciona programas a los cuales tienen como función principal ayudar en el funcionamiento y desarrollo de las labores de la computadora, asi como tambien le dice a la computadora como usar sus propios componentes.

Distribuciones GNU/Linux

Software de aplicación

Son programas que ayudan a los usuarios a realizar cualquier tipo de tarea imaginable, desde un procesador de texto hasta una infinidad de tareas especificas.

Software de aplicación

  • Procesadores de Texto
  • Planillas de cáculo.
  • Graficadores
  • Reproductores de Audio y video.
  • Etc.
